SS_Cache::set_cache_lifetime

Set the cache lifetime for a particular named cache

    Signature

    public function set_cache_lifetime(string $for, [integer $lifetime = 600, [integer $priority = 1]] )

    Parameters

    $for — string
    The name of the cache to set this lifetime for (or 'any' for all backends)
    $lifetime — integer
    The lifetime of an item of the cache, in seconds, or -1 to disable caching
    $priority — integer
    The priority. The highest priority setting is used. Unlike backends, 'any' is not
    special in terms of priority.
