Filesystem::sync

This function ensures the file table is correct with the files in the assets folder.

If a Folder record ID is given, all of that folder's children will be synchronised. If the given Folder ID isn't found, or not specified at all, then everything will be synchronised from the root folder (singleton Folder). See to sync properties of a single database record back to the equivalent filesystem record.
