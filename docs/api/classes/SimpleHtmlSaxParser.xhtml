    public

    • SimpleHtmlSaxParser() — Sets the listener.
    • acceptAttributeToken() — Part of the tag data.
    • acceptEndToken() — Accepts a token from the end tag mode.
    • acceptEntityToken() — A character entity.
    • acceptStartToken() — Accepts a token from the tag mode. If the starting element completes then the element is dispatched and the current attributes set back to empty. The element or attribute name is converted to lower case.
    • acceptTextToken() — Character data between tags regarded as important.
    • createLexer() — Sets up the matching lexer. Starts in 'text' mode.
    • decodeHtml() — Decodes any HTML entities.
    • ignore() — Incoming data to be ignored.
    • normalise() — Turns HTML into text browser visible text. Images are converted to their alt text and tags are supressed.
    • parse() — Runs the content through the lexer which should call back to the acceptors.
