VirtualPageTest::testPublishingAVirtualPageCopiedPublishedContentNotDraftContent

Virtual pages are always supposed to chose the same content as the published source page.

This means that when you publish them, they should show the published content of the source page, not the draft content at the time when you clicked 'publish' in the CMS.
