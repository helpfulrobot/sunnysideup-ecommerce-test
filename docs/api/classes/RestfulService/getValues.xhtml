RestfulService::getValues

Gets set of node values as an array.

When you get to the depth in the hierarchy use node_child_subchild syntax to get the value.

    Signature

    public function getValues(string $xml, [string $collection = NULL, [string $element = NULL]] )

    Parameters

    $xml — string
    The the source xml to parse, this could be the original response received.
    $collection — string
    The name of parent node which wraps the elements, if available
    $element — string
    The element we need to extract the node values.
