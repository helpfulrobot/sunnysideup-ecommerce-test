    public

    • SimpleCookieJar() — Constructor. Jar starts empty.
    • _findFirstMatch() — Finds a matching cookie to write over or the first empty slot if none.
    • _isMatch() — Tests cookie for matching against search criteria.
    • agePrematurely() — Ages all cookies in the cookie jar.
    • getCookieValue() — Reads the most specific cookie value from the browser cookies. Looks for the longest path that matches.
    • restartSession() — Removes expired and temporary cookies as if the browser was closed and re-opened.
    • selectAsPairs() — Uses a URL to sift relevant cookies by host and path. Results are list of strings of form "name=value".
    • setCookie() — Sets an additional cookie. If a cookie has the same name and path it is replaced.
