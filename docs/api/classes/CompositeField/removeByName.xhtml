CompositeField::removeByName

Remove a field from this CompositeField by Name.

The field could also be inside a CompositeField.

    Signature

    public function removeByName(string $fieldName, [boolean $dataFieldOnly = false] )

    Parameters

    $fieldName — string
    The name of the field
    $dataFieldOnly — boolean
    If this is true, then a field will only
    be removed if it's a data field. Dataless fields, such as tabs, will
    be left as-is.
