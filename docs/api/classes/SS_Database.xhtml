    • endSchemaUpdate() — Completes a schema-updated transaction, executing all the schema chagnes.
    • enumValuesForField() — Returns the enum values available on the given field
    • formattedDatetimeClause() — function to return an SQL datetime expression that can be used with the adapter in use used for querying a datetime in a certain format
    • getConnect() — Build the connection string from input
    • getGeneratedID() — Get the autogenerated ID from the previous INSERT query.
    • getLock() — Sets an application-level lock so that no two processes can run at the same time, also called a "cooperative advisory lock".
    • hasField() — Return true if the table exists and already has a the field specified
    • hasTable() — Returns true if the given table exists in the database
    • isActive() — Check if the connection to the database is active.
    • isSchemaUpdating() — Returns true if we are during a schema update.
    • manipulate() — Execute a complex manipulation on the database.
    • prepStringForDB() — Wrap a string into DB-specific quotes. MySQL, PostgreSQL and SQLite3 only need single quotes around the string.
    • query() — Execute the given SQL query.
    • quiet() — Enable supression of database messages.
    • releaseLock() — Remove an application-level lock file to allow another process to run (if the execution aborts (e.g. due to an error) all locks are automatically released).
    • renameField() — Change the database column name of the given field.
    • renameTable() — Rename a table.
    • replace_with_null() — Replaces "\'\'" with "null", recursively walks through the given array.
    • requireField() — Generate the given field on the table, modifying whatever already exists as necessary.
    • requireIndex() — Generate the given index in the database, modifying whatever already exists as necessary.
    • requireTable() — Generate the following table in the database, modifying whatever already exists as necessary.

    • sqlSelectToString() — Returns the SELECT clauses ready for inserting into a query.
    • sqlWhereToString() — Returns the WHERE clauses ready for inserting into a query.
    • supportsLocks() — Determines if the used database supports application-level locks, which is different from table- or row-level locking.
