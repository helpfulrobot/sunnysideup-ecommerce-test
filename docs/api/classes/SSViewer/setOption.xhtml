SSViewer::setOption

Set a global rendering option.

The following options are available: - rewriteHashlinks: If true (the default), <a href="#..."> will be rewritten to contain the current URL. This lets it play nicely with our <base> tag. - If rewriteHashlinks = 'php' then, a piece of PHP script will be inserted before the hash links: "<?php echo $_SERVER['REQUEST_URI']; ?>". This is useful if you're generating a page that will be saved to a .php file and may be accessed from different URLs.
