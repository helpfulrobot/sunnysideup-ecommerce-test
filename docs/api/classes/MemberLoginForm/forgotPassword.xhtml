MemberLoginForm::forgotPassword

Forgot password form handler method.

Called when the user clicks on "I've lost my password". Extensions can use the 'forgotPassword' method to veto executing the logic, by returning FALSE. In this case, the user will be redirected back to the form without further action. It is recommended to set a message in the form detailing why the action was denied.
