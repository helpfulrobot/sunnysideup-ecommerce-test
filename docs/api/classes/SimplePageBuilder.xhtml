    Methods

    public

    • SimplePageBuilder() — Sets the builder up empty.
    • _addContentTagToOpenTags() — Parsed data in tag form. The parsed tag is added to every open tag. Used for adding options to select fields only.
    • _addContentToAllOpenTags() — Any content fills all currently open tags unless it is part of an option tag.
    • _createPage() — Creates an empty page.
    • _createParser() — Creates the parser used with the builder.
    • _hasNamedTagOnOpenTagStack() — Test to see if there are any open tags awaiting closure that match the tag name.
    • _openTag() — Opens a tag for receiving content. Multiple tags will be receiving input at the same time.
    • addContent() — Unparsed, but relevant data. The data is added to every open tag.
    • endElement() — End of element event.
    • free() — Frees up any references so as to allow the PHP garbage collection from unset() to work.
    • parse() — Reads the raw content and send events into the page to be built.
    • startElement() — Start of element event. Opens a new tag.
