TreeDropdownField::__construct

CAVEAT: for search to work properly $labelField must be a database field, or you need to setSearchFunction.

    Signature

    public function __construct(string $name, [string $title = null, [string|array $sourceObject = 'Group', [string $keyField = 'ID', [string $labelField = 'TreeTitle', [bool $showSearch = true]]]]] )

    Parameters

    $name — string
    the field name
    $title — string
    the field label
    $sourceObject — string
    The object-type to list in the tree. This could
    be one of the following:
    - A DataObject class name with the extension.
    - An array of key/value pairs, like a source. In
    this case, the field will act like show a flat list of tree items,
    without any hierarchy. This is most useful in conjunction with
    , for presenting a set of checkboxes in
    a compact view. Note, that all value strings must be XML encoded
    safely prior to being passed in.
    $keyField — string
    to field on the source class to save as the
    field value (default ID).
    $labelField — string
    the field name to show as the human-readable
    value on the tree (default Title).
    $showSearch — object
    enable the ability to search the tree by
    entering the text in the input field.
