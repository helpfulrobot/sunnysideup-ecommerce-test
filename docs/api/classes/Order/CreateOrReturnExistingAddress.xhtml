Order::CreateOrReturnExistingAddress

Returns either the existing one or a new Order Address...

All Orders will have a Shipping and Billing address attached to it. Method used to retrieve object e.g. for $order->BillingAddress(); "BillingAddress" is the method name you can use. If the method name is the same as the class name then dont worry about providing one.
