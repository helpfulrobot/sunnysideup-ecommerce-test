ManyManyList::__construct

Create a new ManyManyList object.

A ManyManyList object represents a list of records that correspond to a many-many relationship. Generation of the appropriate record set is left up to the caller, using the normal methods. Addition arguments are used to support and methods.

    Signature

    public function __construct(string $dataClass, string $joinTable, string $localKey, string $foreignKey, array $extraFields )

    Parameters

    $dataClass — string
    The class of the DataObjects that this will list.
    $joinTable — string
    The name of the table whose entries define the content of this many_many relation.
    $localKey — string
    The key in the join table that maps to the dataClass' PK.
    $foreignKey — string
    The key in the join table that maps to joined class' PK.
    $extraFields — array
    A map of field => fieldtype of extra fields on the join table.
