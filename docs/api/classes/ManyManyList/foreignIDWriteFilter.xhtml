ManyManyList::foreignIDWriteFilter

Return a filter expression for the join table when writing to the join table

When writing (add, remove, removeByID), we need to filter the join table to just the relevant entries. However some subclasses of ManyManyList (Member_GroupSet) modify foreignIDFilter to include additional calculated entries, so we need different filters when reading and when writing
