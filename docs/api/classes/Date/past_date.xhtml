Date::past_date

Return the nearest date in the past, based on day and month.

Automatically attaches the correct year. This is useful for determining a financial year start or end date.

    Parameters

    The number of the month (e.g. 3 is March, 4 is April)
    $fday — integer
    The day of the month
    $fyear — int
    Determine historical value

    Returns
