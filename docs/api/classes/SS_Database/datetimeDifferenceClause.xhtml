SS_Database::datetimeDifferenceClause

function to return an SQL datetime expression that can be used with the adapter in use used for querying a datetime substraction

    Signature

    public function datetimeDifferenceClause(string $date1, string $date2 )

    Parameters

    $date1 — string
    can be either 'now', literal datetime like '1973-10-14 10:30:00' or field name
    e.g. '"SiteTree"."Created"'
    $date2 — string
    to be substracted of $date1, can be either 'now', literal datetime
    like '1973-10-14 10:30:00' or field name, e.g. '"SiteTree"."Created"'

    Returns

    string
    SQL datetime expression to query for the interval between $date1 and $date2 in seconds which
    is the result of the substraction
