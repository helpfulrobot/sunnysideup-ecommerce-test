SS_Database::requireField

Generate the given field on the table, modifying whatever already exists as necessary.

    Signature

    public function requireField(string $table, string $field, array|string $spec )

    Parameters

    $table — string
    The table name.
    $field — string
    The field name.
    $spec — object
    The field specification. If passed in array syntax, the specific database
    driver takes care of the ALTER TABLE syntax. If passed as a string, its assumed to
    be prepared as a direct SQL framgment ready for insertion into ALTER TABLE. In this case you'll
    need to take care of database abstraction in your DBField subclass.
