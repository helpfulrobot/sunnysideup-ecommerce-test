DataObject::getChangedFields

Return the fields that have changed.

The change level affects what the functions defines as "changed": - Level 1 will return strict changes, even !== ones. - Level 2 is more lenient, it will only return real data changes, for example a change from 0 to null would not be included. Example return: <code> array( 'Title' = array('before' => 'Home', 'after' => 'Home-Changed', 'level' => 2) ) </code>

    Signature

    public function getChangedFields([boolean $databaseFieldsOnly = false, [integer $changeLevel = 1]] )
