DataObject::get

Return all objects matching the filter sub-classes are automatically selected and included

    Signature

    public function get([string $callerClass = null, [string $filter = "", [string|array $sort = "", [string $join = "", [string|array $limit = null, [string $containerClass = 'DataList']]]]]] )

    Parameters

    $callerClass — string
    The class of objects to be returned
    $filter — string
    A filter to be inserted into the WHERE clause.
    $sort — string
    A sort expression to be inserted into the ORDER BY clause. If omitted,
    self::$default_sort will be used.
    $join — string
    Deprecated 3.0 Join clause. Use leftJoin($table, $joinClause) instead.
    $limit — object
    A limit expression to be inserted into the LIMIT clause.


    $containerClass — string
    The container class to return the results in.

    Returns

    DataList
