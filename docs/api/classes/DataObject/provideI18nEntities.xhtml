DataObject::provideI18nEntities

Collect all static properties on the object which contain natural language, and need to be translated.

The full entity name is composed from the class name and a custom identifier.

    Signature

    public function provideI18nEntities()

    Returns

    array
    A numerical array which contains one or more entities in array-form.
    Each numeric entity array contains the "arguments" for a _t() call as array values: $entity, $string, $priority, $context.
