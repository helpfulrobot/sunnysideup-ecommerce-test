Form::__construct

Create a new form, with the given fields an action buttons.

    Signature

    public function __construct(Controller $controller, [FieldList $name = null, [FieldList $fields = null, [FieldList $actions = null, [Validator $validator = null]]]] )

    Parameters

    $controller — object
    The parent controller, necessary to create the appropriate form action tag.


    $name — object
    The method on the controller that will return this form object.


    $fields — FieldList
    All of the fields in the form - a {@link FieldList} of {@link FormField} objects.


    $actions — FieldList
    All of the action buttons in the form - a {@link FieldLis} of
    {@link FormAction} objects
    $validator — object
    Override the default validator instance (Default: {@link RequiredFields})
