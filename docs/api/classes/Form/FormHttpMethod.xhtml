Form::FormHttpMethod

Returns the real HTTP method for the form: GET, POST, PUT, DELETE or HEAD.

As most browsers only support GET and POST in form submissions, all other HTTP methods are added as a hidden field "_method" that gets evaluated in . See to get a HTTP method for safe insertion into a <form> tag.
