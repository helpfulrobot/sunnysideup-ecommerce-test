Form::saveInto

Save the contents of this form into the given data object.

It will make use of setCastedField() to do this.

    Signature

    public function saveInto(DataObjectInterface $dataObject, [An $fieldList = null] )

    Parameters

    $dataObject — The
    object to save data into


    $fieldList — object
    optional list of fields to process. This can be useful when you have a
    form that has some fields that save to one object, and some that save to another.
