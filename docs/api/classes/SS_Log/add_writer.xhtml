SS_Log::add_writer

Add a writer instance to the logger.

    Signature

    public function add_writer( $writer, [const $priority = null, [string $comparison = '=']] )

    Parameters

    $writer — object
    Zend_Log_Writer_Abstract instance
    $priority — object
    Priority. Possible values: SS_Log::ERR, SS_Log::WARN or SS_Log::NOTICE


    $comparison — string
    comparison operator. Acts on the integer values of the error
    levels, where more serious errors are lower numbers. By default this is "=", which means only
    the given priority will be logged. Set to "<=" if you want to track errors of *at least*
    the given priority.
