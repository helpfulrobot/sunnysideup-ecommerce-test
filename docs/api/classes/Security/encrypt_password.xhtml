Security::encrypt_password

Encrypt a password according to the current password encryption settings.

If the settings are so that passwords shouldn't be encrypted, the result is simple the clear text password with an empty salt except when a custom algorithm ($algorithm parameter) was passed.

    Signature

    public function encrypt_password(string $password, [string $salt = null, [string $algorithm = null, [Member $member = null]]] )

    Parameters

    $password — string
    The password to encrypt
    $salt — string
    Optional: The salt to use. If it is not passed, but
    needed, the method will automatically create a
    random salt that will then be returned as return value.
    $algorithm — string
    Optional: Use another algorithm to encrypt the
    password (so that the encryption algorithm can be changed over the time).
    $member — object
    Optional


    Returns

    mixed
    Returns an associative array containing the encrypted
    password and the used salt in the form: <code> array( 'password' => string, 'salt' => string, 'algorithm' => string, 'encryptor' => PasswordEncryptor instance ) </code> If the passed algorithm is invalid, FALSE will be returned.
