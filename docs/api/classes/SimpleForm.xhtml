    Methods

    public

    • SimpleForm() — Starts with no held controls/widgets.
    • _addCheckbox() — Adds a checkbox, making it a group on a repeated name.
    • _addRadioButton() — Adds a radio button, building a group if necessary.
    • _createAction() — Combined action attribute with current location to get an absolute form target.
    • _encode() — Creates the encoding for the current values in the form.
    • _setEncodingClass() — Creates the request packet to be sent by the form.
    • _setWidget() — Sets the widget into the form, grouping radio buttons if any.
    • addWidget() — Adds a tag contents to the form.
    • attachLabelBySelector() — Used by the page object to set widgets labels to external label tags.
    • getAction() — Absolute URL of the target.
    • getId() — ID field of form for unique identification.
    • getMethod() — Accessor for method of form submission.
    • getValue() — Extracts current value from form.
    • hasImage() — Test to see if a form has an image control.
    • hasSubmit() — Test to see if a form has a submit button.
    • setDefaultTarget() — Sets the frame target within a frameset.
    • setField() — Sets a widget value within the form.
    • submit() — Simply submits the form without the submit button value. Used when there is only one button or it is unimportant.
    • submitButton() — Gets the submit values for a selected button.
    • submitImage() — Gets the submit values for an image.
