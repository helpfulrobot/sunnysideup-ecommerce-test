Requirements::set_force_js_to_bottom

Set the javascript to be forced to end of the HTML, or use the default.

Useful if you use inline <script> tags, that don't need the javascripts included via Requirements::require();

    Signature

    public function set_force_js_to_bottom(boolean $var )

    Parameters

    $var — boolean
    If true, force the javascripts to be included at the bottom.
