Requirements::themedCSS

Registers the given themeable stylesheet as required.

A CSS file in the current theme path name "themename/css/$name.css" is first searched for, and it that doesn't exist and the module parameter is set then a CSS file with that name in the module is used. NOTE: This API is experimental and may change in the future.

    Signature

    public function themedCSS(string $name, [string $module = null, [string $media = null]] )

    Parameters

    $name — string
    The name of the file - e.g. "/css/File.css" would have
    the name "File".
    $module — string
    The module to fall back to if the css file does not
    exist in the current theme.
    $media — string
    The CSS media attribute.
