ProductGroup::ProductGroupsFromAlsoShowProducts

returns a list of Product Groups that have the products for the CURRENT product group listed as part of their AlsoShowProducts list.

EXAMPLE: You can use the AlsoShowProducts to list products by Brand. In general, they are listed under type product groups (e.g. socks, sweaters, t-shirts), and you create a list of separate ProductGroups (brands) that do not have ANY products as children, but link to products using the AlsoShowProducts many_many relation. With the method below you can work out a list of brands that apply to the current product group (e.g. socks come in three brands - namely A, B and C)
