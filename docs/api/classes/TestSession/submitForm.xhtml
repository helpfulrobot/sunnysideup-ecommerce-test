TestSession::submitForm

Submit the form with the given HTML ID, filling it out with the given data.

Acts on the most recent response. Any data parameters have to be present in the form, with exact form field name and values, otherwise they are removed from the submission. Caution: Parameter names have to be formatted as they are in the form submission, not as they are interpreted by PHP. Wrong: array('mycheckboxvalues' => array(1 => 'one', 2 => 'two')) Right: array('mycheckboxvalues[1]' => 'one', 'mycheckboxvalues[2]' => 'two')

    Signature

    public function submitForm(string $formID, [string $button = null, array $data] )

    Parameters

    $formID — string
    HTML 'id' attribute of a form (loaded through a previous response)
    $button — string
    HTML 'name' attribute of the button (NOT the 'id' attribute)
    $data — array
    Map of GET/POST data.

    Returns

    SS_HTTPResponse
