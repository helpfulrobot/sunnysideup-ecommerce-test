Pelago\Emogrifier::parseCssDeclarationBlock

Parses a CSS declaration block into property name/value pairs.

Example: The declaration block "color: #000; font-weight: bold;" will be parsed into the following array: "color" => "#000" "font-weight" => "bold"

    Signature

    private function parseCssDeclarationBlock(string $cssDeclarationBlock )

    Parameters

    $cssDeclarationBlock — string
    the CSS declaration block without the curly braces, may be empty

    Returns

    array
    the CSS declarations with the property names as array keys and the property values as array values
