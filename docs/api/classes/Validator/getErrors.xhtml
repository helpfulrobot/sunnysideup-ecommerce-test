Validator::getErrors

Returns all errors found by a previous call to {@link validate()}.

The array contains the following keys for each error: - 'fieldName': the name of the FormField instance - 'message': Validation message (optionally localized) - 'messageType': Arbitrary type of the message which is rendered as a CSS class in the FormField template, e.g. <span class="message (type)">. Usually "bad|message|validation|required", which renders differently if framework/css/Form.css is included.
