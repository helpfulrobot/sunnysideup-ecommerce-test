FormField::castedCopy

Returns another instance of this field, but "cast" to a different class.

The logic tries to retain all of the instance properties, and may be overloaded by subclasses to set additional ones. Assumes the standard FormField parameter signature with its name as the only mandatory argument. Mainly geared towards creating *_Readonly or *_Disabled subclasses of the same type, or casting to a . Does not copy custom field templates, since they probably won't apply to the new instance.
