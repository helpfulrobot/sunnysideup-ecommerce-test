UploadField::setValue

Loads the related record values into this field. UploadField can be uploaded in one of three ways:

- By passing in a list of file IDs in the $value parameter (an array with a single key 'Files', with the value being the actual array of IDs). - By passing in an explicit list of File objects in the $record parameter, and leaving $value blank. - By passing in a dataobject in the $record parameter, from which file objects will be extracting using the field name as the relation field. Each of these methods will update both the items (list of File objects) and the field value (list of file ID values).
