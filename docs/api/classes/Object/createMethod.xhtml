Object::createMethod

Add an extra method using raw PHP code passed as a string

    Signature

    protected function createMethod(string $method, string $code )

    Parameters

    $method — string
    the method name
    $code — string
    the PHP code - arguments will be in an array called $args, while you can access this object
    by using $obj. Note that you cannot call protected methods, as the method is actually an external
    function
