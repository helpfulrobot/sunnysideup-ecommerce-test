Object::has_extension

Return TRUE if a class has a specified extension.

This supports backwards-compatible format (static Object::has_extension($requiredExtension)) and new format ($object->has_extension($class, $requiredExtension))

    Signature

    public function has_extension(string $classOrExtension, [string $requiredExtension = null, [boolean $strict = false]] )

    Parameters

    $classOrExtension — string
    if 1 argument supplied, the class name of the extension to
    check for; if 2 supplied, the class name to test
    $requiredExtension — string
    used only if 2 arguments supplied
    $strict — boolean
    if the extension has to match the required extension and not be a subclass
