Object::remove_extension

Remove an extension from a class.

Keep in mind that this won't revert any datamodel additions of the extension at runtime, unless its used before the schema building kicks in (in your _config.php). Doesn't remove the extension from any instances which are already created, but will have an effect on new extensions. Clears any previously created singletons through to avoid side-effects from stale extension information.
