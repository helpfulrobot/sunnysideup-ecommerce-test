Object::useCustomClass

This class allows you to overload classes with other classes when they are constructed using the factory method {@link Object::create()}

    Signature

    public function useCustomClass(string $oldClass, [string $newClass = false, [bool $strong = false]] )

    Parameters

    $oldClass — string
    the class to replace
    $newClass — string
    the class to replace it with
    $strong — object
    allows you to enforce a certain class replacement under all circumstances. This is used in
    singletons and DB interaction classes
