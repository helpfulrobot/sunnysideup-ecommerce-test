Injector::updateSpec

Update the configuration of an already defined service

Use this if you don't want to register a complete new config, just append to an existing configuration. Helpful to avoid overwriting someone else's changes updateSpec('RequestProcessor', 'filters', '%$MyFilter')

    Signature

    public function updateSpec(string $id, [string $property = true, [mixed $value = true, [boolean $append = true]]] )

    Parameters

    $id — string

    The name of the service to update the definition for
    $property — string

    The name of the property to update.
    $value — mixed

    The value to set
    $append — boolean

    Whether to append (the default) when the property is an array
