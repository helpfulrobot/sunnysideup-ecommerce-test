Injector::instantiate

Instantiate a managed object

Given a specification of the form array( 'class' => 'ClassName', 'properties' => array('property' => 'scalar', 'other' => '%$BeanRef') 'id' => 'ServiceId', 'type' => 'singleton|prototype' ) will create a new object, store it in the service registry, and set any relevant properties Optionally, you can pass a class name directly for creation To access this from the outside, you should call ->get('Name') to ensure the appropriate checks are made on the specific type.

    Signature

    protected function instantiate(array $spec, [string $id = null, [string $type = null]] )

    Parameters

    $spec — array

    The specification of the class to instantiate
    $id — string

    The name of the object being created. If not supplied, then the id will be inferred from the
    object being created
    $type — string

    Whether to create as a singleton or prototype object. Allows code to be explicit as to how it
    wants the object to be returned
