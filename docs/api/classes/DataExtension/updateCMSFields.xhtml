DataExtension::updateCMSFields

This function is used to provide modifications to the form in the CMS by the extension. By default, no changes are made. {@link DataObject->getCMSFields()}.

Please consider using to globally add formfields to the record. The method should just be used to add or modify tabs, or fields which are specific to the CMS-context. Caution: Use to add fields.
