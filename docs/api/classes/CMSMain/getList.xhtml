CMSMain::getList

Returns the pages meet a certain criteria as {@see CMSSiteTreeFilter} or the subpages of a parent page defaulting to no filter and show all pages in first level.

Doubles as search results, if any search parameters are set through .

    Signature

    public function getList(array $params, [integer $parentID = 0] )

    Parameters

    $params — array
    Search filter criteria
    $parentID — integer
    Optional parent node to filter on (can't be combined with other search criteria)

    Returns

    SS_List

    Errors/Exceptions

    InvalidArgumentException
    if invalid filter class is passed.
