HTMLText::Summary

Create a summary of the content. This will be some section of the first paragraph, limited by $maxWords. All internal tags are stripped out - the return value is a string

This is sort of the HTML aware equivilent to Text#Summary, although the logic for summarising is not exactly the same

    Signature

    public function Summary([integer $maxWords = 50, [integer $flex = 15, [string $add = '...']]] )

    Parameters

    $maxWords — integer
    Maximum number of words to return - may return less, but never more. Pass -1 for no limit
    $flex — integer
    Number of words to search through when looking for a nice cut point
    $add — string
    What to add to the end of the summary if we cut at a less-than-ideal cut point

    Returns

    string
    A nice(ish) summary with no html tags (but possibly still some html entities)
