    • $disabled — array
      Remembers the filepaths of all cleared Requirements through {@link clear()}.
    • $force_js_to_bottom — boolean
      Force the javascripts to the bottom of the page, even if there's a <script> tag in the body already
    • $javascript — array
      Paths to all required .js files relative to the webroot.
    • $suffix_requirements — bool
      Do we want requirements to suffix onto the requirement link tags for caching or is it disabled. Getter / Setter available through {@link Requirements::set_suffix_requirements()}

    • $write_header_comment — boolean
      Setting for whether or not a file header should be written when combining files.
    • $write_js_to_body — boolean
      Put all javascript includes at the bottom of the template before the closing <body> tag instead of the <head> tag.
