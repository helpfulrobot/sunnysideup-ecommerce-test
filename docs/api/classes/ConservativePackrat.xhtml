\ConservativePackrat

Conservative Packrat will only memo-ize a result on the second hit, making it more memory-lean than Packrat, but less likely to go exponential that Parser. Because the store logic is much more complicated this is a net loss over Parser for many simple grammars.

  • Author: Hamish Friedlander
