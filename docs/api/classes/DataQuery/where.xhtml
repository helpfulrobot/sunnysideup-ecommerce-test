DataQuery::where

Append a WHERE clause to this query.

There are two different ways of doing this: <code> // the entire predicate as a single string $query->where("\"Column\" = 'Value'"); // multiple predicates as an array $query->where(array("\"Column\" = 'Value'", "\"Column\" != 'Value'")); </code>
