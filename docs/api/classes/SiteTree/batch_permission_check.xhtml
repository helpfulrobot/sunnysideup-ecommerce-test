SiteTree::batch_permission_check

This method is NOT a full replacement for the individual can*() methods, e.g. {@link canEdit()}.

Rather than checking (potentially slow) PHP logic, it relies on the database group associations, e.g. the "CanEditType" field plus the "SiteTree_EditorGroups" many-many table. By batch checking multiple records, we can combine the queries efficiently. Caches based on $typeField data. To invalidate the cache, use or set the $useCached property to FALSE.
