SiteTree::can_edit_multiple

Get the 'can edit' information for a number of SiteTree pages.

    Signature

    public function can_edit_multiple(array $ids, [int $memberID = true, [bool $useCached = true]] )

    Parameters

    $ids — array
    An array of IDs of the SiteTree pages to look up.
    $memberID — int
    ID of member.
    $useCached — object
    Return values from the permission cache if they exist.


    Returns

    array
    A map where the IDs are keys and the values are booleans stating whether the given
    page can be edited.
