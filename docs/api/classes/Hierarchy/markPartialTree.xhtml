Hierarchy::markPartialTree

Mark a segment of the tree, by calling mark().

The method performs a breadth-first traversal until the number of nodes is more than minCount. This is used to get a limited number of tree nodes to show in the CMS initially. This method returns the number of nodes marked. After this method is called other methods can check isExpanded() and isMarked() on individual nodes.
