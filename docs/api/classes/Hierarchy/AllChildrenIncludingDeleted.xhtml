Hierarchy::AllChildrenIncludingDeleted

Return all children, including those that have been deleted but are still in live.

Deleted children will be marked as "DeletedFromStage" Added children will be marked as "AddedToStage" Modified children will be marked as "ModifiedOnStage" Everything else has "SameOnStage" set, as an indicator that this information has been looked up.
