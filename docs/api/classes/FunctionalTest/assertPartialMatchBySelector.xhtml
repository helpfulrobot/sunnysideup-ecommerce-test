FunctionalTest::assertPartialMatchBySelector

Assert that the most recently queried page contains a number of content tags specified by a CSS selector.

The given CSS selector will be applied to the HTML of the most recent page. The content of every matching tag will be examined. The assertion fails if one of the expectedMatches fails to appear. Note: &nbsp; characters are stripped from the content; make sure that your assertions take this into account.
