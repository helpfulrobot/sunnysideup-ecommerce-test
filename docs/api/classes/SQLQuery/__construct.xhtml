SQLQuery::__construct

Construct a new SQLQuery.

    Signature

    public function __construct([string $select = "*", array $from, array $where, array $orderby, array $groupby, array $having, array $limit] )

    Parameters

    $select — string
    An array of SELECT fields.
    $from — array
    An array of FROM clauses. The first one should be just the table name.
    $where — array
    An array of WHERE clauses.
    $orderby — array
    An array ORDER BY clause.
    $groupby — array
    An array of GROUP BY clauses.
    $having — array
    An array of HAVING clauses.
    $limit — array
    A LIMIT clause or array with limit and offset keys
