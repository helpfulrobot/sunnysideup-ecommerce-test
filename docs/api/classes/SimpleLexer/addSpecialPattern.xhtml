SimpleLexer::addSpecialPattern

Adds a pattern that has a special mode. Acts as an entry and exit pattern in one go, effectively calling a special parser handler for this token only.

@param string $pattern Perl style regex, but ( and ) lose the usual meaning. @param string $mode Should only apply this pattern when dealing with this type of input. @param string $special Use this mode for this one token. @access public
