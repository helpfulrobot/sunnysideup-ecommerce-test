SimpleLexer::addPattern

Adds a token search pattern for a particular parsing mode. The pattern does not change the current mode.

@param string $pattern Perl style regex, but ( and ) lose the usual meaning. @param string $mode Should only apply this pattern when dealing with this type of input. @access public
