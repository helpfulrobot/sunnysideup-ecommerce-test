SimpleLexer::addEntryPattern

Adds a pattern that will enter a new parsing mode. Useful for entering parenthesis, strings, tags, etc.

@param string $pattern Perl style regex, but ( and ) lose the usual meaning. @param string $mode Should only apply this pattern when dealing with this type of input. @param string $new_mode Change parsing to this new nested mode. @access public
