    Methods

    public

    • SimplePage() — Parses a page ready to access it's contents.
    • _addLink() — Adds a link to the page.
    • _extractResponse() — Extracts all of the response information.
    • _getUrlFromLink() — Converts a link tag into a target URL.
    • _isFormElement() — Tests to see if a tag is a possible form element.
    • _isLoadingFrames() — Test to see if in the middle of reading a frameset.
    • _linkIsAbsolute() — Test to see if link is an absolute one.
    • _noResponse() — Sets up a missing response.
    • _setBase() — Sets the base url for the page.
    • _setTitle() — Sets the title tag contents.
    • acceptFormEnd() — Closes the most recently opened form.
    • acceptFormStart() — Opens a form. New widgets go here.
    • acceptFrame() — Takes a single frame tag and stashes it in the current frame set.
    • acceptFramesetEnd() — Closes the most recently opened frameset.
    • acceptFramesetStart() — Opens a frameset. A frameset may contain nested frameset tags.
    • acceptLabelEnd() — Closes the most recently opened label.
    • acceptLabelStart() — Opens a label for a described widget.
    • acceptPageEnd() — Marker for end of complete page. Any work in progress can now be closed.
    • acceptTag() — Adds a tag to the page.
    • clearFrameFocus() — Clears the frame focus. Does nothing for a leaf page.
    • expandUrl() — Expands expandomatic URLs into fully qualified URLs.
    • getAuthentication() — Accessor for last Authentication type. Only valid straight after a challenge (401).
    • getBaseUrl() — Base URL if set via BASE tag page url otherwise @return SimpleUrl Base url.
    • getField() — Accessor for a form element value within a page.
    • getFormById() — Finds a held form by the form ID. A way of identifying a specific form when we have control of the HTML code.
    • getFormByImage() — Finds a held form by image using a selector.
    • getFormBySubmit() — Finds a held form by button label. Will only search correctly built forms.
    • getFrameFocus() — Accessor for current frame focus. Will be false as no frames.
    • getFrames() — Fetches a list of loaded frames.
    • getFrameset() — Accessor for frame name and source URL for every frame that will need to be loaded. Immediate children only.
    • getHeaders() — Accessor for raw headers of page.
    • getMethod() — Original request method.
    • getMimeType() — Accessor for current MIME type.
    • getRaw() — Accessor for raw text of page.
    • getRealm() — Accessor for last Authentication realm. Only valid straight after a challenge (401).
    • getRequest() — Original request as bytes sent down the wire.
    • getRequestData() — Original request data.
    • getResponseCode() — Accessor for HTTP response code.
    • getText() — Accessor for plain text of page as a text browser would see it.
    • getTitle() — Accessor for parsed title.
    • getTransportError() — Accessor for last error.
    • getUrl() — Original resource name.
    • getUrlById() — Accessor for a URL by the id attribute.
    • getUrls() — Accessor for a list of all links.
    • getUrlsByLabel() — Accessor for URLs by the link label. Label will match regardess of whitespace issues and case.
    • hasFrames() — Test for the presence of a frameset.
    • setField() — Sets a field on each form in which the field is available.
    • setFrameFocus() — Sets the focus by name. Always fails for a leaf page.
    • setFrameFocusByIndex() — Sets the focus by index. The integer index starts from 1.
