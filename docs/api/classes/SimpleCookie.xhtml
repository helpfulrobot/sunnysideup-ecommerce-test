    Methods

    public

    • SimpleCookie() — Constructor. Sets the stored values.
    • _fixPath() — Adds a trailing and leading slash to the path if missing.
    • _truncateHost() — Extracts just the domain part that determines a cookie's host validity.
    • agePrematurely() — Ages the cookie by the specified number of seconds.
    • getExpiry() — Accessor for expiry.
    • getHost() — Accessor for the truncated host to which this cookie applies.
    • getName() — Accessor for name.
    • getPath() — Accessor for path.
    • getValue() — Accessor for value. A deleted cookie will have an empty string for this.
    • isExpired() — Test to see if cookie is expired against the cookie format time or timestamp.
    • isSecure() — Accessor for the secure flag.
    • isValidHost() — Test for a cookie being valid for a host name.
    • isValidPath() — Tests a path to see if the cookie applies there. The test path must be longer or equal to the cookie path.
    • setHost() — Sets the host. The cookie rules determine that the first two parts are taken for certain TLDs and three for others. If the new host does not match these rules then the call will fail.
