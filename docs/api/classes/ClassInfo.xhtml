\ClassInfo

Provides introspection information about the class tree.

It's a cached wrapper around the built-in class functions. SilverStripe uses
class introspection heavily and without the caching it creates an unfortunate
performance hit.

    Methods

    public

    • allClasses() — Wrapper for classes getter.
    • ancestry() — Returns the passed class name along with all its parent class names in an array, sorted with the root class first.
    • baseDataClass() — Returns the root class (the first to extend from DataObject) for the passed class.
    • classImplements() — Returns true if the given class implements the given interface
    • classes_for_file() — Get all classes contained in a file.
    • classes_for_folder() — Returns all classes contained in a certain folder.
    • dataClassesFor() — Returns an array of the current class and all its ancestors and children which have a DB table.
    • exists()
    • getValidSubClasses() — Returns the manifest of all classes which are present in the database.

    • subclassesFor() — Returns a list of classes that inherit from the given class.
    • table_for_object_field() — Returns the table name in the class hierarchy which contains a given field column for a {@link DataObject}. If the field does not exist, this will return null.
