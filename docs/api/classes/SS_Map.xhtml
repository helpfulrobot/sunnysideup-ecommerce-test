    public

    • __construct() — Construct a new map around an SS_list.
    • count() — Returns the count of items in the list including the additional items set through {@link SS_Map::push()} and {@link SS_Map::unshift}.
    • getIterator() — Returns an SS_Map_Iterator instance for iterating over the complete set of items in the map.
    • keys() — Return all the keys of this map.
    • offsetExists()
    • offsetGet()
    • offsetSet() — Sets a value in the map by a given key that has been set via {@link SS_Map::push()} or {@link SS_Map::unshift()}
    • offsetUnset() — Removes a value in the map by a given key which has been added to the map via {@link SS_Map::push()} or {@link SS_Map::unshift()}
    • push() — Pushes an item onto the end of the map.
    • setKeyField() — Set the key field for this map.
    • setValueField() — Set the value field for this map.
    • toArray() — Return an array equivalent to this map.
    • unshift() — Unshift an item onto the start of the map.
    • values() — Return all the values of this map.
