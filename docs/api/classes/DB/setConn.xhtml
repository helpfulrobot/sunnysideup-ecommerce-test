DB::setConn

Set the global database connection.

Pass an object that's a subclass of SS_Database. This object will be used when is called.

    Signature

    public function setConn(SS_Database $connection, [string $name = 'default'] )

    Parameters

    $connection — The
    connecton object to set as the connection.


    $name — string
    name to give to this connection. If you omit this argument, the connection
    will be the default one used by the ORM. However, you can store other named connections to
    be accessed through DB::getConn($name). This is useful when you have an application that
    needs to connect to more than one database.
