LeftAndMain::EditFormTools

Renders a panel containing tools which apply to the currently displayed edit form.

The main difference to is that the panel is displayed within the element structure of the form panel (rendered through ). This means the panel will be loaded alongside new forms, and refreshed upon save, which can mean a performance hit, depending on how complex your panel logic gets. Any form fields contained in the returned markup will also be submitted with the main form, which might be desired depending on the implementation details.
