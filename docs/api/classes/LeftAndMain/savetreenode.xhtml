LeftAndMain::savetreenode

Update the position and parent of a tree node.

Only saves the node if changes were made. Required data: - 'ID': The moved node - 'ParentID': New parent relation of the moved node (0 for root) - 'SiblingIDs': Array of all sibling nodes to the moved node (incl. the node itself). In case of a 'ParentID' change, relates to the new siblings under the new parent.
