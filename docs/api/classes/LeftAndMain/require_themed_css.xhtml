LeftAndMain::require_themed_css

Register the given "themeable stylesheet" as required.

Themeable stylesheets have globally unique names, just like templates and PHP files. Because of this, they can be replaced by similarly named CSS files in the theme directory.

    Signature

    public function require_themed_css(String $name, [String $media = null] )

    Parameters

    $name — object
    The identifier of the file. For example, css/MyFile.css would have the identifier "MyFile"


    $media — object
    Comma-separated list of media-types (e.g. "screen,projector")
