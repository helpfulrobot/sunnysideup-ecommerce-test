Director::absoluteURL

Turns the given URL into an absolute URL.

By default non-site root relative urls will be evaluated relative to the current request.

    Signature

    public function absoluteURL(string $url, [bool $relativeToSiteBase = false] )

    Parameters

    $url — string
    URL To transform to absolute
    $relativeToSiteBase — object
    Flag indicating if non-site root relative urls should be
    evaluated relative to the site BaseURL instead of the current url.

    Returns

    string
    The fully qualified URL
