SilverStripeServiceConfigurationLocator::configFor

Retrieves the config for a named service without performing a hierarchy walk

    Signature

    protected function configFor(string $name )

    Parameters

    $name — string
    Name of service

    Returns

    mixed
    Returns either the configuration data, if there is any. A missing config is denoted
    by a value of either null (there is no direct config assigned and a hierarchy walk is necessary) or false (there is no config for this class, nor within the hierarchy for this class).
