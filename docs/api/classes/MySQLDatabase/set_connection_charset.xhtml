MySQLDatabase::set_connection_charset

Sets the character set for the MySQL database connection.

The character set connection should be set to 'utf8' for SilverStripe version 2.4.0 and later. However, sites created before version 2.4.0 should leave this unset or data that isn't 7-bit safe will be corrupted. As such, the installer comes with this set in mysite/_config.php by default in versions 2.4.0 and later.
