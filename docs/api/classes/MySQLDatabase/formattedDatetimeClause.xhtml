MySQLDatabase::formattedDatetimeClause

function to return an SQL datetime expression that can be used with MySQL used for querying a datetime in a certain format

    Signature

    public function formattedDatetimeClause(string $date, string $format )

    Parameters

    $date — string
    to be formated, can be either 'now', literal datetime like '1973-10-14 10:30:00' or
    field name, e.g. '"SiteTree"."Created"'
    $format — string
    to be used, supported specifiers:
    %Y = Year (four digits)
    %m = Month (01..12)
    %d = Day (01..31)
    %H = Hour (00..23)
    %i = Minutes (00..59)
    %s = Seconds (00..59)
    %U = unix timestamp, can only be used on it's own

    Returns

    string
    SQL datetime expression to query for a formatted datetime
