CompositeDBField::writeToManipulation

Add the custom internal values to an INSERT or UPDATE request passed through the ORM with {@link DataObject->write()}.

Fields are added in $manipulation['fields']. Please ensure these fields are escaped for database insertion, as no further processing happens before running the query. Use . Ensure to write NULL or empty values as well to allow unsetting a previously set field. Use for the appropriate type.
