\SS_Sortable

Additional interface for {@link SS_List} classes that are sortable.

All methods in this interface are immutable - they should return new instances with the sort
applied, rather than applying the sort in place

    Synopsis

    interface SS_Sortable {
    • // methods
    • public bool canSortBy()
    • public SS_Sortable sort()
    • public SS_Sortable reverse()
    }

    Methods

    public

    • canSortBy() — Returns TRUE if the list can be sorted by a field.
    • reverse() — Return a new instance of this list based on reversing the current sort.
    • sort() — Return a new instance of this list that is sorted by one or more fields. You can either pass in a single field name and direction, or a map of field names to sort directions.
